# generalized_Bekic

This repository contains the artifact for the submission "An Inductive Proof of the generalized Bekic Principle and its Formalization in Isabelle".

There are two files:

Util.thy

Locales.thy

Not everything in Util.thy is relevant, but parts of it is necessary for Locales.thy to compile.

One should be able to load the two files directly into Isabelle without issues.

The main theorems are contained in Locales.thy. The generalized Bekic principle is general_Bekic.
